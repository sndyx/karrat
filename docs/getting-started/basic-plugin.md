# Basic Plugin

After [adding the required dependencies](setup.md) to your project, it's time to
 make your main file.

```
example-plugin
├── build.gradle.kts
├── gradle
├── gradlew
├── gradlew.bat
├── settings.gradle.kts
└── src
    ├── main
    │   ├── kotlin
    │   │   └── com
    │   │       └── example
    │   │           └── ranks
    │   │               └── ExamplePlugin.kt
    │   └── resources
    └── test
```
